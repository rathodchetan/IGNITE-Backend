# IGNITE-Backend
Begin with 
```npm i```

include .env file in same folder as index.js with environment variables

ex: Change Values according to your database configuration values.
  ``` 
  DB_USER = "postgres"
  DB_HOST = "localhost"
  DB_DATABASE = "ignite"
  DB_PASSWORD = "postgres123"
  DB_PORT = "5432" 
  ```
